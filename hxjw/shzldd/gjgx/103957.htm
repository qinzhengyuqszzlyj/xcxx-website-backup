莘城二年级组育儿沙龙活动——育儿心得分享
发布日期:2018-01-04    作者：莘城学校   点击率：


活动目的

通过家长沙龙活动，对于小学生，良好的习惯的养成教育是最重要的，

它远比获得知识要重要的多。而习惯好的孩子，家长的正确引导也起着十分重要的作用，因此充分调动起家长的育儿积极性，努力配合教育工作。

 

活动主题

（一）家长如何教育孩子

（二）如何塑造孩子良好的品格。

（三）如何纠正孩子的不良习惯

整个内容围绕以上这几个主题展开，很多语言比较书面化，只是给大家一个借鉴，最好的是结合自己身边的教育子女的例子，理论与实际结合阐述。

 

活动方式

交流、论坛，研讨，家长对在孩子习惯养成教育上的好的做法，聆听家长的心声。 

活动时间

2017年3月4日周五晚6:30-7:30

活动地点

二年级（5）班教室

 

前

期

准

备

 

 

1、  提前一周告知家长沙龙活动的时间。（让家长做好充足的会前准备，以便能及时参加会议） 

2、  每班选5位家长代表参加（最好是家委会成员）

3、  每班选一位家长代表结合自身孩子教育情况，分享一些孩子的教育方法及建议，为家长提供优秀的家教经验。（每位时长5分钟左右）

 

 

会

序

2、  主持人致欢迎词，感谢各位家长的到来。介绍本次会议的环节。

家委会会议发言记录

 

二（1）班 刘铭轩家长 石黎黎

分享好习惯的培养

小朋友已经渐渐适应小学的生活，家长在入学前有自己学习，知道好习惯的重要性。

1-      父母和孩子成为好朋友。孩子放学后很愿意跟父母分享在学校的一天，性格比较开朗。有助于父母了解孩子的学习动态

2-      父母给孩子做行为习惯的表率。孩子在写作业的时间家长从来不看电视。

3-      作业由孩子独立完成，家长只是负责最后的检查和签名，聪明地“偷懒“。孩子碰到不会的地方先让孩子自己尝试解决。

4-      安排每天睡觉前孩子有专门的听故事的时间。借由故事的形式让孩子接触美好的文字。

5-      每个孩子花期不同，让我们静静地等待他们各自盛开。

 

 

 

二（2）班 于文溪家长 于涛

兼谈家校互动

1-      孩子的特点是爱读书。即使是比较难的书或者在比较吵闹的环境都能安静地认真地读。

2-      行为习惯的养成要点：家长以身作则，率先垂范；及时表扬，引导，软硬兼施；持之以恒，坚持就是胜利；形成环境（家庭环境与学校环境相结合）

3-      对于家长的挑战：家长要打破固有的思维模式；要建立平等和谐的亲子关系；自身要有良好的行为习惯

4-      家校互动：执行学校的规范与准则，家校一起努力。

 

 

 

二（3）班 吴鑫宸家长 吴慧

爸爸承认自己光说不做，但希望跟大家分享反面的案例

1-      从小给孩子做了不少早教，但是进入小学之后发现孩子有些厌学情绪。给孩子压力太大对孩子有害无益。

2-      辅导孩子学习的时候没有考虑孩子的实际接受能力，没有遵循孩子的思维能力。

3-      应该多与老师沟通，与孩子一起进步。

4-      这位父亲看到孩子有厌学情绪的时候积极与老师沟通，缓解了自己的焦虑的同时也总结出了接下来的亲子教育思路：善于发现孩子的潜力；激发孩子的动力，做自己喜欢做的事情；培养孩子的活力，要积极锻炼身体。

5-      家长需要不断地学习和思考。不能只能注重孩子的成绩，更要注重孩子行为习惯和人格的养成。首先要培养孩子乐观向上的性格和心态；然后是与人交流的能力。

 

二（4）班 朱籽韵家长 黄骏

我家有女初长成

小孩特别，班级最小，比较懵懂，学习不是很主动

1-      家长要和老师进行有效的沟通，在学习方法和心态上与学校行为合力。

2-      家庭教育也很重要，注重温故知新；寓教于乐，学问来自生活，同一件事情可以从语文，数学，英语等不同的方面来引导。

3-      用心陪伴，静待花开

 

 

二（5）班 罗兆轩家长 罗鹏

希望抛砖引玉

孩子在班上年龄最小，比较吃力

爸爸根据孩子的特点，建立了适合孩子的培养计划

1-      多跟老师了解孩子的状况，回家再针对性纠正

2-      没有在外多报兴趣班，孩子应该优先完成学校的课业。

3-      孩子放学去晚托班，作业基本能按实完成。多看到孩子的进步，与自己比。

4-      注重孩子的体育锻炼，已经参加游泳和跆拳道多年。

5-      孩子的信心比较弱，爸爸希望孩子在其它方面的闪光点可以增强孩子的信心。

6-      教之道，贵以专。家长要坚持对孩子的培养。养育孩子要耐心，要找到适合自己孩子的养育办法。

 

 

二（6）班 宋思颖家长

孩子也是年龄比较小，性格比较内向。

1-      家长注重孩子的教育，常常找别人取经。但发现很多不适合自己的孩子。

2-      每个孩子都不同，孩子的教育不能一概而论，家长应该根据孩子的特点进行引导

3-      注重孩子预习与复习的习惯。培养自己的学习能力。

4-      对待孩子粗心造成的错误不让步。

5-      家长对孩子的成长要有耐心。尊重孩子的心智发育时间。

 

二（7）班 陈文薇家长 阮琳燕

大宝已经在7年级，没有良好学习习惯的孩子学习很辛苦

1-      对孩子要有耐心，暴力解决不了问题。

2-      妈妈对孩子的引导很重要，妈妈一定要跟孩子有好的关系。

 3特殊孩子特殊对待。

 

老师总结发言：

感谢家长们对学校工作的支持和帮助。

学校的读书节，书法特色教育都是希望给孩子养成好习惯助力。

孩子将来的竞争不是智商的竞争，而是阅读能力的竞争。号召家长以身作则，推进孩子阅读能力的培养。

莘城学校准备推进必读100部书的活动。希望家长一起推荐。
