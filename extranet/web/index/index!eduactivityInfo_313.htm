闵行区“小学低年级主题式综合活动课程”研讨活动通知

活动时间: 2022-09-29 13:00

对象：小学

地点：闵行区航华第二小学（航中路355号）

年级：小学

学科：主题式综合活动课程

内容：

为了整区推进“小学低年级主题式综合活动课程”的建设与实施，将继续开展系列现场展示研讨活动，本次活动具体安排如下：

 

主题：办童年学校 还童真本色

流程：

（一）13:00~13:15  参观签到（由一楼大厅走向二楼彼得潘剧场）

（二）13:15~13:30    学校汇报

（三）13:35~14:50    综合活动展示

                    一年级《汤团圆圆》（执教：谢佩燕）

         二年级《皮影戏真有趣》（执教：孙娟）

（四）14:55~15:25     区校联合教研（兼新版指导纲要和综合活动设计模板解读培训）   

（五）15:30~16:00     专家点评、领导讲话


出席与附言：线下：区小学低年级主题式综合活动课程试点校项目负责人（每校限一人） 线上：各公办小学课程部主任（教导主任），市区级试点校实验教师，其他面上学校教师以及区级研修班全体学员等。（直播链接另行通知）

备注：1.严格遵守防疫要求：①全程佩戴口罩；②配合测量体温；③出示24小时核酸阴性报告； 2.校内没有停车位，参与活动的老师请绿色出行（有停车需求的教师可以前往航北路和航中路的社会车辆公共停车位，停车费自理）
