2022学年第一学期第10周周二初三英语网络教研活动

活动时间: 2022-11-01 13:00

对象：初中

地点：闵行区云录播课堂教学评价系统，网址：http://ypk.ct-edu.com.cn/ylb/home/index

年级：九年级,初中

学科：英语

内容：

一、 活动主题：

1、初三写作专题训练（二）：应用文写作2（2022年上海市中考英语作文题辅导）

2、“优化课堂活动，培养学生学习能力”主题研究公开课展示

 二、观课平台：区云录播平台, 网址http://ypk.ct-edu.com.cn/ylb/home/index

三、观课、评课时间：

2022年11月1日（周二）下午13：00 — 11月11日（周五）下班前

四、活动流程及任务要求：

1、观看文绮中学丁梦莹老师（女）的一节初三英语应用文写作课。

课题：Creating an Activity Room —2022年上海市中考作文题辅导

 (网上播出时间：线下授课2022/10/18  13:40 - 14:20)，在网上完成 “量表评课并提交评分”；在网上完成“评论交流并提交评论”。)

2、观看莘松中学宋怡怡老师（女）一节初三英语阅读课。

课题：9AU5 The human brain More practice

 (网上播出时间：线下授课2021/9/28  13:50 - 14:30)，在网上完成 “量表评课并提交评分”；在网上完成“评论交流并提交评论”。)

 3、请各校初三英语备课组老师在备课组长带领下做好组内研讨并有活动纪要（以后调研时要检查的）。同时做好后续跟进工作，如：相同的课题可以根据各校的学情，在各自的班级进行教学实践。

 

 


出席与附言：请各校初三英语备课组老师在备课组长带领下按照时间节点完成工作任务，谢谢大家参与。
