问题二：和孩子沟通的四个障碍是什么？

 

1号 莫奥扬

 

主要是：1、缺乏足够的耐心2、批评多、表扬少 3、有时用命令式口吻 4、不理解孩子的感受和需求

 

3号 谢浩然

 

（1）一件事情要说很多次但是效果都不是很好。

   （2）对于“什么时间干什么事情”孩子始终都很难执行到位。

   （3）对于“做什么事情都要有方法”这个要求确实不是很喜欢动脑筋解决问题。

   （4）对于“保护眼睛”的原则性经常遗忘。

 

6号 胡子辰

 

沟通方式单一

说教式、灌输式沟通是被广泛采用的一种方法，在与孩子沟通时，讲得口干舌燥，而孩子则是低头受训，这种沟通方式很难使孩子接受自己的观点。

沟通过程专断化 
 常把自己当作"教育者"，单方向进行沟通的，希望孩子都能听进去，只顾自己"畅所欲言"。其实孩子根本一字没听进。

沟通中家长意见不一 
 家里除了父母，还有外公外婆，在与孩子沟通时，老人们常会有不同意见并当孩子面表达，造成孩子不清楚谁对谁错。父母的权威性大打折扣。

沟通中缺乏鼓励和赏识 
 孩子在沟通时得到的认可较少，缺乏共同的话题，沟通意愿不强。

 

8号 李沛霖

 

✓ 不理解、体谅孩子 

✓ 不会欣赏孩子 

✓ 家长太强势 

✓ 和孩子之间没有共同话题

 

 

9号 李维轩

 

   （1）感觉现在不管说什么，孩子总是听不进去，左耳朵进右耳朵出。心平气和的说几乎没有任何效果，必须要大声、严厉，才可能有一点效果。

   （2）对待学习的要求不同，家长总是希望孩子能够将所学过的知识都能真正掌握，打捞基础，但孩子总是抱着应付的态度，应付完老师的作业，便再也不想多做点什么。

（3）信息沟通不畅，家长不能了解孩子的真实想法，孩子无法理解家长的要求。

（4）孩子兴趣点与家长的希望想背离，影响到双方沟通的情绪和效果。比如孩子对游戏特别有兴趣，但家长对这个兴趣是反感甚至是有很多担忧的。

 

10号 刘畅

 

四个障碍主要是立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

 

11号 刘冠迪

从自己和孩子沟通中出现的问题看，主要存在以下四个障碍：

1) “镇压式”的家长：要多与孩子沟通，在生活中有些琐事会让自己不自觉地放在很强势的地位，总想着孩子按自己想法去做，导致沟通一开始就不是处在平等的地位。

2) 不理解孩子的感受和需求：在生活中，抓住每个细节，每件小事与孩子沟通，放下姿态，站在孩子的角度与孩子谈心，真正了解孩子的想法。

3) 批评多，表扬少：做错事就该批评，但我会跟他一起沟通，这样才能避免再次犯错。同时，我也会发现一些他做对的地方进行表扬，表达自己的欣赏，让孩子提高自己的自信心。让他有正确的世界观和价值观。

4) 过高的期望，不合理的目标设置：放平心态，水到渠成，最自然的也是最健康的。作为父亲，我希望他快乐健康的成长，没有太高的期望强加在他身上。

 

 

12号 吕奕成

 

1. 不尊重孩子。

首先，孩子再小，也是个人。其次，沟通，就是双方在平等地位上的交流。不尊重孩子，根本不可能产生真正的沟通。

2. 不给孩子自由。

如果不给孩子适当的自由，他没有自己的意志，孩子也很难真正表达出自己内心真正想要所处的话。

3. 不懂得聆听。

只有认真、耐心的听懂孩子的话，真正理解他的需要，才能有的放矢，针对性孩子的疑惑或者问题进行对话和引导。

4. 不懂得沟通的技巧。

沟通，也需要技巧。不好的沟通技巧，可能“事倍功半”；而高效的沟通技巧，可以使“事半功倍”。

14号 乔治平

 

（1） “镇压式”的家长

两代人平等沟通。大人多从孩子角度想孩子的问题。

（2）理解孩子的感受和需求

和孩子是知心朋友，无话不谈。

（3）批评多，表扬少

表扬“吹捧”多。批评婉转，多用反问句：“为什么不……呢？”

（4）过高的期望，不合理的目标设置

主要期望是健康，懂得快乐，有爱好。

15号 盛明智

 

四个障碍主要是立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

16号 杨颢宸

● 不理解

● 不作为

● 不合作

● 不主动

17号 谢沈珈

1. 由于父母们总是以“完美”的尺度来衡量孩子。站在他们的角度，以他们的思维来考虑问题，就容易产生不理解和体谅孩子

2. 不会欣赏孩子，欣赏，不是简单的夸赞，而是发自内心地对一样东西的珍视。我们的父母没有欣赏过我们，所以我们不会欣赏自己，进而不会欣赏自己的孩子，其实不欣赏也是一种创伤，这对孩子的身心发展是非常不利的 

3. 现在，很多事业上很成功的父母，工作给他们带来强大的压力与责任，所以他们会投注很多精力在工作上，对待孩子，甚至采用工作上的方式，采用高压政策。强势的父母，通常愿采取简单粗暴方式，希望在一顿怒火与责骂之下，孩子会神奇般地变好了。不过，这只是他们的一厢情愿，家长太强势会造成亲子关系较生疏，宝宝缺少自己主见，可能导致孩子叛逆

4. 年龄不一样，态度和想法不一样，不能互相理解， 所以容易产生代沟和孩子之间没有共同话题。

18号 汤锦麟

沟通的四个障碍主要是：1,没有沟通。2,育儿识少。3,年龄差距大，感觉缺少一份童心。4生活压力大。

19号 沈迎卿

 1.唠叨：唠叨是一种不好习惯，唠叨中夹杂着批评，指责，重翻旧账。孩子听到就会不耐烦，甚至恼火，认为家长不理解他们。

   2.说教：父母对孩子的说教往往是高高在上的，不是平等基础上的交流，这样就很难照顾到孩子的情绪。这样教育下的孩子他们的情绪得不到关照。要么在说教中爆发，要么在说教中压抑。

   3.建议：父亲通常别把自己的想法强加到孩子身上，这让孩子很不舒服，甚至容易产生叛逆的情绪。

   4.命令：这是最要命的，孩子再小也是独立的个体，他们也有自己的想法。

20号 田惠汀

四个障碍主要是立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

21号 张欣蕾

 

四个障碍是：唠叨，说教，建议，命令

实际中很难避开这些

22号 张钰雯

1、不理解    2、不作为   3、不合作   4、不主动

23号 周沁睿

   因为在家一直扮红脸的关系，在沟通时缺乏倾听，缺少鼓励，在批评孩子时，会造成孩子的紧张，无反映，迟钝。

24号 朱丽雯

（1）不能相互理解，没有共同话题。

   （2）家长的强势让孩子不能敢于说出自己的想法。

   （3）缺少赏识和肯定孩子的优点和长处，过多的关注了孩子的缺点。   

   （4）孩子的兴趣爱好与家长希望的不符。

25号 张晟嘉

主要是唠叨、说教、建议和命令

26号 朱籽韵

1. 不尊重孩子

以大欺小，总觉得我是家长，我是大人，我吃过的盐比孩子喝过的水都多，还没开口，心理上就已经压着孩子了，沟通之前已经有问题了。

2. 不清楚孩子真实的全面的状况

平时家长大多忙于工作， 在没搞清楚孩子的情况，优点、缺点、喜好、习性，从只言片语，从单一的事件或角度，就匆匆得出结论，先入为主地与孩子交流，给孩子造成无谓的委屈和心理压力，久而久之就会失去孩子的信任，任何事情都不愿意跟家长交流，这对孩子的成长会造成极大的伤害，而不仅仅是体现在亲情上。

3. 做一个好的聆听者远比一副智者的样子要好

孩子有孩子的想法和思维，加上时代不同，不能用我们小时候的经验和经历来衡量他们现在的世界，只有耐心地倾听才能比较全面地知道孩子真实的想法，不要轻易打断孩子，匆忙下结论。

4. 不懂得欣赏孩子

作为家长要努力关注孩子各方面的发展，既要关注不好的，也要关注好的，要给予孩子多一些鼓励，培养孩子的信心，失去信心就会失去兴趣，失去兴趣一切就会变成被动的，索然无味的，任何事情都干不好。

27号 邹菲桐

（1） 对待学习的要求孩子不是很理解，父亲常常是希望孩子掌握方法、牢固基础，但孩子总是以完成任务为目标，不能体会父母的要求与目的。

（2）对孩子兴趣与课外教育培养不自信，孩子难以讲清楚自己的兴趣爱好，父母也拿捏不准，在有些课外辅导班上，影响到双方沟通的情绪和效果。比如孩子对学而思有畏惧情绪，但家长觉得这个很重要，有时产生不愉快。

（3）教育期望值不一致，尽管孩子是一个比较自律、积极上进的孩子，但每每与父母的期望还有不小差距，交流过程中产生冲突。

(4)自己比较粗心，小时候与女孩打交道不多，不了解女孩的需要。真正了解女性是从与孩子母亲及孩子的交流中学习体会的。孩子也有很深的体会，常说“爸爸好是很好，就是不知道我想要什么，不能主动买给我”。

28号 陈家玉

目前和孩子不存在沟通障碍。

29号 李念祖

立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

30号 宋昀皙

四个障碍是：唠叨，说教，建议，命令。

● 因此父亲要换位思考和少唠叨，偶尔唠叨不要夹入批评和指责；

● 关注孩子情绪，不要说教；

● 只提供参考性建议，不强加想法；

● 摒弃威胁的命令，理解孩子的个体独立。

立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

33号 倪艺文

四个障碍主要是：

1. 立场不同导致的互相不理解、恐惧造成的信息沟通不畅、命令式教育带来的单向指令、望子成龙心切引发的急切心理与孩子缓慢成长的差距。

2. 家长和孩子之间没有共同的话题。

3. 家长不会欣赏孩子身上的闪光点。

4. 不知道孩子真正需要的是什么。

34号 潘歆怡

与孩子沟通时自身的情绪不对；不能够耐心的倾听孩子的诉求；打断孩子的言语；与孩子的意见有冲突。

35号 潘予辰

从自己和孩子沟通中出现的问题看，主要存在以下四个障碍：

1) “镇压式”的家长：同孩子沟通中，不自觉地就把自己放在很强势的地位，居高临下，导致沟通一开始就不是处在平等的地位。

2) 不理解孩子的感受和需求：我们往往以大人的思维惯式来看待孩子的所作所为，导致我们不能理解孩子的真正想法和需求。家长觉得孩子不可理喻而孩子觉得委屈万分，这样如何能做好沟通呢？

3) 批评多，表扬少：我们应该不吝啬对孩子表达自己的欣赏，让孩子提高自己的自信心。当然，要控制好度，不能把孩子的不足也夸成优点，这样就变成无原则的溺爱了。

4) 过高的期望，不合理的目标设置：平时对孩子提出这样那样的要求，往往忽略了孩子正常的成长规律，不免有拔苗助长之嫌。放平心态，水到渠成，最自然的也是最健康的。

有时批评出口容易，表扬少。

2、家长比较严格：

   导致孩子有任何问题都敢跟父母沟通

   欣赏，不是简单的夸赞，而是发自内心地对一样东西的珍视。

4、和孩子之间没有共同话题

   如果想知道孩子的内心世界，首先要了解孩子们所处的这个时代，父母只有了解了孩子们生活的时代才能进一步了解孩子。再有就是要多关心孩子，作为家长最起码应该知道孩子的最好的朋友是谁。他们经常在一起都谈论什么话题。和孩子们多做沟通，及时了解他们的想法，多从孩子的角度出发，我想孩子不会不喜欢跟你交流，也不会出现楼主说的那种问题。

4) 过高的期望，不合理的目标设置：放平心态，水到渠成，最自然的也是最健康的。作为父亲，我希望他快乐健康的成长，没有太高的期望强加在他身上。

 

 

12号 吕奕成

 

1. 不尊重孩子。

首先，孩子再小，也是个人。其次，沟通，就是双方在平等地位上的交流。不尊重孩子，根本不可能产生真正的沟通。

2. 不给孩子自由。

如果不给孩子适当的自由，他没有自己的意志，孩子也很难真正表达出自己内心真正想要所处的话。

3. 不懂得聆听。

只有认真、耐心的听懂孩子的话，真正理解他的需要，才能有的放矢，针对性孩子的疑惑或者问题进行对话和引导。

4. 不懂得沟通的技巧。

沟通，也需要技巧。不好的沟通技巧，可能“事倍功半”；而高效的沟通技巧，可以使“事半功倍”。

和孩子是知心朋友，无话不谈。

平时家长大多忙于工作， 在没搞清楚孩子的情况，优点、缺点、喜好、习性，从只言片语，从单一的事件或角度，就匆匆得出结论，先入为主地与孩子交流，给孩子造成无谓的委屈和心理压力，久而久之就会失去孩子的信任，任何事情都不愿意跟家长交流，这对孩子的成长会造成极大的伤害，而不仅仅是体现在亲情上。

孩子有孩子的想法和思维，加上时代不同，不能用我们小时候的经验和经历来衡量他们现在的世界，只有耐心地倾听才能比较全面地知道孩子真实的想法，不要轻易打断孩子，匆忙下结论。

4. 不懂得欣赏孩子

作为家长要努力关注孩子各方面的发展，既要关注不好的，也要关注好的，要给予孩子多一些鼓励，培养孩子的信心，失去信心就会失去兴趣，失去兴趣一切就会变成被动的，索然无味的，任何事情都干不好。

2) 不理解孩子的感受和需求：我们往往以大人的思维惯式来看待孩子的所作所为，导致我们不能理解孩子的真正想法和需求。家长觉得孩子不可理喻而孩子觉得委屈万分，这样如何能做好沟通呢？
